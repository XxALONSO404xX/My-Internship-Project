Connection Error

The application could not connect to the Vite development server. This typically happens when:

  • The Vite server isn't running
  • The Vite server is running on a different port
  • There's a network issue preventing the connection

Try the following:

  1. Ensure Vite is running by executing npx vite in the project directory
  2. Check the terminal for the Vite server URL and port
  3. Restart the Electron application
